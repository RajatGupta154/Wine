
.csv file format:

        name, brand, type, country of origin, year of made, volume,price, stock, description, recomendation, image filename + file extension

All fields are mandatory except the stock, description, recommendation and image.

If you wish to skip one field mark with ,,.

The description and recommendation should be written between " ".
